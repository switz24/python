simple bot for me and my friends

functionality includes:
1. announcing that somebody joined a voicecall
2. play youtube videos trough youtube
3. generate qr codes with a given link
4. joining and leaving voicecalls on command

prefix for commands: !
list of commands:
!help
join   / makes the bot join your current voiceall
leave  / makes the bot leave the current voicecall
play <yt-link> / plays the audio from a youtube channel in the current voicecall
pause / pauses the audio
resume / resumes the audio
stop / stops the audio playback alltogether
qr <link> / generates a qr code for the given link
swift / dont use this

note!
pls only add a space bar then link for every command with a <link> attribute.
For example:
!qr www.github.com / will generate a qr code for the github homepage :D
